(R) Data Mart

★ Built and compared linear regression and random forest models on 550,000+ Walmart transactions to analyze purchase behavior and forecast spending.

Project Overview

This project analyzes over 550,000 Walmart transactions to understand customer purchasing behavior and build predictive models for purchase amount. After preparing and exploring the data, two modeling approaches were implemented: a linear regression model and a random forest model. Model performance was evaluated using cross-validation and a held-out validation set, with metrics including R², RMSE, and MAE.

## Convert DataTypes to Factors
```{r}
Walmart <- Walmart |> 
  mutate(Gender = as.factor(Gender),
         Age = as.factor(Age),
         Occupation = as.factor(Occupation),
         City_Category = as.factor(City_Category),
         Stay_In_Current_City_Years = as.factor(Stay_In_Current_City_Years),
         Marital_Status = as.factor(Marital_Status),
         Product_Category = as.factor(Product_Category)) |> 
  select(Purchase,Age,Gender,City_Category,Stay_In_Current_City_Years,Occupation,Marital_Status,Product_Category)
```
