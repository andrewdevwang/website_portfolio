(R) Airline Tweets

★ Processed 14,000+ tweets and transformed these tweets using topic modeling into an intuitive world cloud.

Project Overview

This project analyzes Twitter data from United Airlines, using text mining and unsupervised learning in R. The goal of this is to identify frequently discussed themes, uncover hidden patterns in customer conversations, and map these trends onto a clear visualization. The workflow consists of: data cleaning → exploratory text analysis → topic modeling → visualization.
