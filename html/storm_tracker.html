(Java) Storm Tracker

★ Automated a data parser that transformed over 5,000 structureless plain text hurricane records into structured tables that allow retireval by attribute

Project Overview

StormChaser is program that reads raw hurricane observation records, groups them into structured data based on intensity using wind and pressure, filters for major hurricanes, and sorts the severe storms by intensity.

Section 7: Ordering Storms

To order storms by their severity, a bubble sort algorithm is used. This compares adjacent storms and swaps them if they are not in order.

public static void Sort(Storm[] stormList, int n) {

    int pass = 0, k, switches;
    Storm temp;

    switches = 1;

    while(switches != 0) {

        switches = 0;
        pass++;

        for(k = 0; k < n - pass; k++) {

            if(stormList[k].getCategory() < stormList[k+1].getCategory()) {

                temp = stormList[k];
                stormList[k] = stormList[k+1];
                stormList[k+1] = temp;

                switches = 1;
            }
        }
    }
}
